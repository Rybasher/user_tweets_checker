# Tweets checker

## Setup project

- fill environments files
- run project

## Fill environment
1. go to server and run
    - > cp .env.example .env
2. provide TWITTER_BEARER_TOKEN
3. provide USER_ID
4. go to client and run 
    - > cp .env.example .env

## Run with docker

> docker-compose up --build

## Start without docker


1. Go to server folder and run
     - > npm i
     - > npm run start 
2. Go to client folder and run
    - > npm i
    - > npm run start
